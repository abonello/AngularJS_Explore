One-Way Binding

Code:

In AngularJS, the code is:

var app = angular.module("app", []);
app.controller("oneWay", ["$scope", function($scope) {
    $scope.a = 10;
    $scope.b = 20;
}]);

In the HTML, the code is:

Using an Expression:
The value of <strong>a</strong> is {{a}}, and the value of <strong>b</strong> is {{b}}.

Using ng-bind:
in the HTML, the code is:
The value of <strong>a</strong> is <span ng-bind="a"></span>, and the value of <strong>b</strong> is <span ng-bind="b"></span>.
