One-Way Binding

Code:

In AngularJS, the code is:

var app = angular.module("app", []);
app.controller("oneWay", ["$scope", function($scope) {
    $scope.a = 10;
    $scope.b = 20;
}])

In the HTML, the code is:

Using an Expression:
The value of <strong>a</strong> is {{a}}, and the value of <strong>b</strong> is {{b}}.

Using ng-bind:
in the HTML, the code is:
The value of <strong>a</strong> is <span ng-bind="a"></span>, and the value of <strong>b</strong> is <span ng-bind="b"></span>.

One-way binding Using Expressions

The value of a is {{a}}, and the value of b is {{b}}.

One-way binding Using ng-bind

The value of a is , and the value of b is .

This can be useful in a situation where the UI is used only to display data generated elsewhere. Here the user is not interacting with the data, ie. the data cannot be changed by the user. It is Read-Only. However the data will be updated in realtime as soon as it is modified in the models.
Note that ng-bind can be used with non-input elements only.

One-Time Binding

Code:

In AngularJS, the code is:

app.controller("oneTime", ["$scope", function($scope) {
    $scope.c = 10;
    $scope.d = 20;

    $scope.doubleValues = function() {
        $scope.c = $scope.c * 2;
        $scope.d = $scope.d * 2;
    };

    $scope.sumValues = function() {
        $scope.sum = $scope.c + $scope.d;
    };
}])

In the HTML, the code is:

This part changes

The value of <strong>c</strong> is {{c}}, and the value of <strong>d</strong> is {{d}}.

This part does not change

Notice the double colon in the expression.
The <em>initial value</em> of <strong>c</strong> is {{::c}}, and the <em>initial value</em> of <strong>d</strong> is {{::d}}.

One-way binding (Using Expressions) - updates

The value of c is {{c}}, and the value of d is {{d}}.

One-Time binding (Using Expressions) - does not update

This uses double colon ::

The initial value of c is {{::c}}, and the initial value of d is {{::d}}.

Notice how the values of c and d are not changing as we only want the initial value and that is what the code is saying.

The sum of the current c and current d is:

Sum = {{sum}}

Note that sum does not update when the values of c and d is doubled. You need to run the method to sum the new values by clicking the appropriate button above.

This can be useful in a situation where we need to keep tab of certain initial values to see how much they have changed.

Two-Way Binding

Code:

In AngularJS, the code is:

app.controller("twoWay", ["$scope", function($scope) {
  $scope.e = 10;
  $scope.f = 15;

  $scope.addNumbers = function() {
    $scope.sum = parseInt($scope.e) + parseInt($scope.f);
    //The parsing is required as we will be getting input from user as a string.
  };
}])

In the HTML, the code is:

Display of current values of e and f

These update in real-time as the user types into the input boxes below.

Input of new values for e and f

New value for e: <input type="text" ng-model="e"/>
New value for f: <input type="text" ng-model="f"/>

Display of the sum
